# Specification Requirements Document
# AlreadyExists — iOS App/ MacOS App
Version: 1.0 | Status: Draft | Owner: Product

---

## Problem Statement
iOS developers do not have an easy way to track if an application name already exists without multiple search across the app store. This app helps to search for an application by name.

## User Stories

### iOS Developer
- As a developer, I want to open this application and be able to type in the name of an application and find if the name is taken.

#### Acceptance Criteria
1. Given a user enters a name this application will use the safest and most appropriate means to find out if that application exists by name or ot and inform the user of such.
2. Given a description of what an app is intended to do this application use the safest and most appropriate means to find out if that application exists by name or ot and inform the user of such.
